## 🎓 Learning Objectives
According to TryHackMe, in this room you will learn:
- **Web Enumeration:** Identifying hidden files and directories using tools like Gobuster and inspection of source code/robots.txt.
- **Exploiting Web Servers:** Finding credentials and using them to gain access to a web portal.
- **Command Injection:** Executing system commands through a vulnerable web application field.
- **Privilege Escalation:** Identifying weak sudo permissions to gain root access.

## 🧠 Knowledge Required
- **Basic Linux CLI:** Understanding commands like `ls`, `cat`, and `sudo`.
- **Web Basics:** Understanding how HTTP, source code, and directory structured files work.
- **Tools:** Familiarity with Nmap and Gobuster is helpful but can be learned during the room.
