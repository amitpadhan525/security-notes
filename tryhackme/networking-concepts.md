## 🎓 Learning Objectives
According to TryHackMe, in this room you will learn:
- **OSI Model:** Deep dive into the ISO network model.
- **Addressing & Routing:** Understanding IP addresses, subnets, and how data is routed.
- **Protocols & Ports:** Differentiating between TCP/UDP and how port numbers work.
- **Hands-on Interaction:** Using Telnet to connect to open ports from the command line.
- **Encapsulation:** How data is prepared for transmission.

## 🧠 Knowledge Required
- **Foundational:** Basic familiarity with terms like IP address and TCP port is recommended.
