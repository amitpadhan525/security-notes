Tools Nmap

Nmap

Network Mapper — the industry-standard tool for network discovery and security auditing. Essential for every pentest engagement to map targets, discover services, detect OS versions, and run vulnerability scripts.

Basics & Target Syntax

Legal: Only scan systems you have explicit written permission to scan. Unauthorized port scanning may be illegal in your jurisdiction.

bash — Target Formats
nmap 192.168.1.1              # single IP
nmap 192.168.1.1-254          # IP range
nmap 192.168.1.0/24           # CIDR subnet
nmap 10.0.0.1 10.0.0.2        # multiple hosts
nmap -iL targets.txt          # read from file
nmap scanme.nmap.org          # hostname

Scan Types

-sS

SYN Scan (Stealth)

Default for root. Sends SYN, gets SYN-ACK (open) or RST (closed). Never completes handshake — rarely logged by applications. Requires root.

-sT

TCP Connect

Full 3-way handshake. Default for non-root. Slower and more detectable — logged by applications. Used when raw sockets unavailable.

-sU

UDP Scan

Scans UDP ports (DNS:53, SNMP:161, DHCP:67/68). Very slow — no response means open|filtered, ICMP unreachable means closed.

-sA

ACK Scan

Maps firewall rules rather than open ports. Determines filtered vs unfiltered — doesn't confirm if a port is actually open.

-sN/-sF/-sX

NULL / FIN / Xmas

Stealth scans that bypass some firewalls. Work against BSD/Unix — unreliable against Windows (always shows closed).

-sn

Ping Scan

Host discovery only — no port scan. Fast for large ranges. Sends ICMP echo, TCP SYN to 443, TCP ACK to 80.

Port & Service Options

bash — Port & Detection Flags
# Port specification
nmap -p 80 target              # single port
nmap -p 80,443,8080 target     # multiple ports
nmap -p 1-1000 target          # range
nmap -p- target                # ALL 65535 ports
nmap -F target                 # fast top-100 ports
nmap --top-ports 1000 target   # top 1000 (default)

# Service version detection
nmap -sV target                # probe services for version
nmap -sV --version-intensity 9 target  # max accuracy

# OS detection (requires root)
nmap -O target
nmap -O --osscan-guess target  # aggressive guessing

# Aggressive all-in-one (OS + version + scripts + traceroute)
nmap -A target

# Script scan (default NSE scripts)
nmap -sC target

CTF tip: Always run -p- to find ALL ports. Many CTF machines run SSH on 2222, HTTP on 8000, etc. You will miss them with the default top-1000 scan.

NSE — Key Scripts

bash — NSE Usage
# Run specific script
nmap --script http-title target
nmap --script smb-vuln-ms17-010 target     # EternalBlue
nmap --script ftp-anon target              # anon FTP check
nmap --script ssl-heartbleed target        # Heartbleed

# Script categories
nmap --script vuln target       # all vuln detection scripts
nmap --script auth target       # auth bypass checks
nmap --script brute target      # brute force
nmap --script discovery target  # service enumeration

# Wildcard matching
nmap --script "smb-vuln-*" target    # all SMB vuln scripts
nmap --script "http-*" target        # all HTTP scripts

# With arguments
nmap --script snmp-info --script-args snmpcommunity=public target
Script What It Does
smb-vuln-ms17-010 Detect EternalBlue / WannaCry Windows SMB RCE
smb-vuln-ms08-067 Classic Windows XP/2003 NetAPI RCE
http-shellshock Detect Shellshock in CGI scripts
ssl-heartbleed Detect OpenSSL Heartbleed vulnerability
ftp-anon Check if anonymous FTP login is allowed
http-enum Enumerate common web paths and files
http-robots.txt Retrieve robots.txt disallowed paths
dns-zone-transfer Attempt DNS zone transfer (AXFR)
ssh-auth-methods List accepted SSH authentication methods
rdp-enum-encryption Enumerate RDP encryption settings
snmp-info Enumerate SNMP device information
ldap-rootdse Active Directory enumeration via LDAP

Timing, Output & Evasion

bash — Timing & Output
# Timing (-T0 slowest/stealthiest, -T5 fastest/noisiest)
nmap -T0 target    # Paranoid (IDS evasion — very slow)
nmap -T1 target    # Sneaky
nmap -T3 target    # Normal (default)
nmap -T4 target    # Aggressive (CTF standard)
nmap -T5 target    # Insane (may miss results)

# Performance control
nmap --min-rate 5000 target    # minimum 5000 packets/sec
nmap --max-retries 1 target    # only retry probes once
nmap --host-timeout 30s target # give up on host after 30s

# Output formats
nmap -oN scan.txt target       # normal human-readable
nmap -oX scan.xml target       # XML (for tools like Metasploit)
nmap -oG scan.gnmap target     # grepable
nmap -oA scan target           # ALL formats simultaneously

# Firewall evasion
nmap -f target                       # fragment packets (8-byte)
nmap -D RND:10 target                # 10 random decoy IPs
nmap --source-port 53 target         # spoof DNS source port
nmap --spoof-mac 0 target            # random spoofed MAC

Full Pentest Workflow

bash — CTF / Pentest Workflow
TARGET=10.10.10.100

# 1. Quick ping sweep
nmap -sn 10.10.10.0/24

# 2. Fast initial scan
nmap -T4 -F $TARGET

# 3. Full port scan
nmap -p- -T4 --min-rate 5000 $TARGET -oN allports.txt

# 4. Deep scan on found ports
PORTS=$(grep open allports.txt | cut -d/ -f1 | tr '\n' , | sed 's/,$//')
nmap -sV -sC -O -p$PORTS $TARGET -oN detailed.txt

# 5. Vuln scan
nmap --script vuln -p$PORTS $TARGET -oN vulns.txt

# 6. Service-specific scripts
nmap --script smb-vuln-* -p445 $TARGET
nmap --script http-enum,http-title -p80,443,8080 $TARGET
nmap --script ftp-anon -p21 $TARGET
