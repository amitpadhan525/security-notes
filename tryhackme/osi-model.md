---

## 🎓 Learning Objectives
According to TryHackMe, in this room you will learn:
- **OSI Model Layers:** Detailed functions of each of the 7 layers (Physical to Application).
- **Data Flow:** How data travels through these layers.
- **Practical Application:** Using an interactive game to test knowledge of the model.
- **Packets & Frames:** Differentiating between data units at various layers.

## 🧠 Knowledge Required
- **Foundational Room:** No prior networking experience is required, though Introductory Networking is a good companion.

---

## 🔍 Task Breakdown
1. **Task 1: What is the OSI Model?** - Introduction to the concept.
2. **Task 2-8: Layers 1-7** - Individual deep dives into each layer.
3. **Task 9: Practical - OSI Game** - Hands-on challenge to apply the model.
4. **Task 10: Packets & Frames** - Understanding data encapsulation.
