According to TryHackMe, in this room you will learn:
- **Hash Identification:** Learning to recognize different types of hashes (MD5, SHA1, etc.).
- **Password Cracking:** Using tools like John the Ripper and Hashcat to retrieve plain-text from hashes.
- **Cracking Challenges:** Practicing with various levels of difficulty.

## 🧠 Knowledge Required
- **Basic Linux CLI:** Familiarity with the terminal.
- **Tools:** Basic understanding of password cracking tools is helpful.
