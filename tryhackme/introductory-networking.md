## 🎓 Learning Objectives
According to TryHackMe, in this room you will learn:
- **Networking Theory:** Understanding the OSI and TCP/IP models.
- **Networking Tools:** How to use common tools like Ping, Traceroute, WHOIS, and Dig.
- **Encapsulation:** How data is wrapped in different layers of the network stack.

## 🧠 Knowledge Required
- **Introductory Level:** This is a foundational room; no advanced prior knowledge is required.
