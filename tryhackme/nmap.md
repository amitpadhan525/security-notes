## 🎓 Learning Objectives
According to TryHackMe, in this room you will learn:
- **Discover Live Hosts:** Identify active devices on a network.
- **Service Identification:** Find running services (e.g., SSH, web servers).
- **Port Scan Types:** Distinguish between various scan types.
- **Service Version Detection:** Extract details about running service versions.
- **Timing Control:** Manage performance and stealth by adjusting scan speed.
- **Output Formatting:** Save and format scan results for reporting or automation.

## 🧠 Knowledge Required
- **TCP/IP Model:** Understanding related concepts and protocols.
- **Networking Fundamentals:** Familiarity with concepts taught in prerequisite rooms (Networking Concepts, Essentials, etc.).

## 🔍 Task Breakdown
1. **Task 1: Introduction** - Overview of Nmap and scanning objectives.
2. **Task 2: Subnetworks and IP Addresses** - Understanding network segmentation.
3. **Task 3: TCP and UDP Ports** - How ports work in communication.
4. **Task 4: Nmap Live Host Discovery** - Using `-sn` and other flags to find hosts.
5. **Task 5: Nmap Basic Port Scans** - Performing initial scans.
6. **Task 6: Nmap Advanced Scanning Options** - Stealth and complex scans.
7. **Task 7: Nmap Service and OS Detection** - Identifying software and operating systems.
8. **Task 8: Nmap Scripting Engine (NSE)** - Automating tasks and vulnerability scanning.
9. **Task 9: Nmap Saving Output** - Exporting results in various formats.
10. **Task 10: Summary** - Conclusion and key takeaways.
