## 🎓 Learning Objectives
According to TryHackMe, in this room you will learn:
- **Linux Background:** Origins of Linux and why it is used in security.
- **Basic Commands:** Using `ls`, `cd`, and `cat` to navigate and read files.
- **Filesystem Interaction:** Understanding the Linux directory structure.
- **Searching for Files:** Using tools like `find` and `grep`.
- **Shell Operators:** Using `&`, `&&`, `>`, and `>>` for command execution and redirection.

## 🧠 Knowledge Required
- **Foundational Room:** No prior Linux experience is required.
