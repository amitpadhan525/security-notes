# RootMe - TryHackMe

## 🚩 Room Information
- **Link:** [TryHackMe - RootMe](https://tryhackme.com/room/r00tme)
- **Difficulty:** Easy
- **Category:** Web Exploitation / Privilege Escalation

---

## 🎓 Learning Objectives
According to TryHackMe, in this room you will learn:
- **Reconnaissance:** Scanning and enumeration techniques using tools like Nmap and Gobuster.
- **Web Exploitation:** Gaining initial access through file upload bypasses and getting a reverse shell.
- **Privilege Escalation:** Identifying and exploiting SUID binaries in a Linux environment to gain root access.

## 🧠 Knowledge Required
- **Basic Linux CLI:** Navigation, file permissions, and common commands.
- **Basic Security Tools:** Understanding how Nmap and directory brute-forcing tools work.
- **Reverse Shells:** Basic concept of connecting back to an attacker machine.

---

## 🔍 Task Breakdown
1. **Task 1: Deploy the machine** - Initial setup and connection.
2. **Task 2: Reconnaissance** - Enumerating ports and web directories.
3. **Task 3: Getting a shell** - Finding an upload page, bypassing filters, and executing a shell.
4. **Task 4: Privilege Escalation** - Finding SUID bits and using Python for root access.
