## 🎓 Learning Objectives
According to TryHackMe, in this room you will learn:
- **CPU Architecture:** Understanding what a CPU is and how its architecture (x86) is structured.
- **Registers:** Exploring main components and how data is stored/used by the CPU.
- **Memory Management:** How memory works and the importance of the stack layout.
- **Component Interaction:** How data flows between memory and the processor.
