## 🎓 Learning Objectives
According to TryHackMe, in this room you will learn:
- **Navigating Windows:** Exploring the desktop, taskbar, and file explorer.
- **Managing Files:** Basic file operations and organization in a Windows environment.
- **System Tools:** Introduction to Control Panel, Task Manager, and other essential utilities.
- **Configuration & Security:** Basic system settings and security features.

## 🧠 Knowledge Required
- **Foundational Room:** No prior technical experience with Windows administration is required.

1. **Task 1: Introduction** - Overview of the Windows operating system.
